# 🛒 Flux — E-Commerce Platform

A full-stack e-commerce web application built with React, Node.js, Express, and MongoDB.

## Features
- User registration and login (JWT authentication)
- Product catalog with search and category filtering
- Shopping cart with quantity management
- Checkout and order placement
- Admin panel to manage products and orders

## Tech Stack
- Frontend: React, React Router, Axios
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Authentication: JWT + bcrypt

## Setup Instructions
1. Clone: `git clone https://github.com/Junto0010/Flux.git`
2. Backend: `cd backend && npm install && npm run dev`
3. Frontend: `cd frontend && npm install && npm start`
4. Set MONGO_URI and JWT_SECRET in backend/.env
